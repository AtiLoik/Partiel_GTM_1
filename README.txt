Pour installer l'application web:
1) Copier/coller le contenu du fichier Partiel1.sql dans votre base de donnée
2) Installer le dossier Partiel_GTM_1 sur votre serveur APACHE
3) Modifiez le fichier connexion.php avec vos identifiants a la base de données


Informations utiles:
- L'identifiant et le mot de passe du directeur sont son nom de famille et le mot de passe entré dans la base de données
- Toutes les fonctionalités sont accessibles depuis le tableau des eleves.
